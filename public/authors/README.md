# Author Images

Place author portrait images in this folder with the following filenames:

## Required Images

- `john-donne.jpg` - John Donne
- `robert-frost.jpg` - Robert Frost
- `maya-angelou.jpg` - Maya Angelou
- `william-shakespeare.jpg` - William Shakespeare
- `sara-teasdale.jpg` - Sara Teasdale
- `walt-whitman.jpg` - Walt Whitman
- `langston-hughes.jpg` - Langston Hughes
- `joyce-kilmer.jpg` - Joyce Kilmer
- `percy-shelley.jpg` - Percy Bysshe Shelley
- `rudyard-kipling.jpg` - Rudyard Kipling
- `christina-rossetti.jpg` - Christina Georgina Rossetti
- `mary-frye.jpg` - Mary Elizabeth Frye
- `edgar-poe.jpg` - Edgar Allan Poe
- `elizabeth-browning.jpg` - Elizabeth Barrett Browning
- `william-henley.jpg` - William Ernest Henley
- `dylan-thomas.jpg` - Dylan Thomas
- `emily-dickinson.jpg` - Emily Dickinson
- `william-blake.jpg` - William Blake
- `default.jpg` - Default image for unlisted authors

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 400x400px minimum (square aspect ratio)
- **Quality**: High quality portraits, preferably historical photographs or paintings
- **Rights**: Ensure images are in public domain or you have permission to use them

## Finding Images

Good sources for public domain author portraits:
- Wikimedia Commons (commons.wikimedia.org)
- Library of Congress Digital Collections
- Project Gutenberg
- Public domain image repositories

## Note

The app uses Next.js Image component with responsive sizing, so larger images will be automatically optimized.
